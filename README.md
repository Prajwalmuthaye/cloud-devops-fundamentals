# cloud-devops-fundamentals

Amazon Web Services (AWS) is a cloud computing platform that provides on-demand IT infrastructure and services over the internet using a pay-as-you-go pricing model.

Instead of buying physical servers, companies rent resources like:
1.Virtual machines
2.Storage
3.Databases
4.Networking
5.Security services

# Cloud Computing Basics
Definition:
Cloud computing is the delivery of computing services over the internet with:
On-demand access
Scalability
Elasticity
Metered billing
# Cloud Service Models
1.IaaS (Infrastructure as a Service)
You manage OS, applications, and configuration.
Example:
Amazon EC2
2.PaaS (Platform as a Service)
AWS manages infrastructure and OS.
Example:
AWS Elastic Beanstalk
3.SaaS (Software as a Service)
Fully managed software accessed via browser.
Example:
Google Workspace

# Availability Zones (AZs)
Multiple isolated data centers within a region
Used for high availability
# Edge Locations
Used for content delivery (CDN)
# Core Services: 
1.Compute Services
Amazon EC2 – Virtual servers
AWS Lambda – Serverless execution

2.Storage Services
Amazon S3 – Object storage
Amazon EBS – Block storage
Amazon EFS – Shared file system

3.Networking Services
Amazon VPC – Virtual network
Security Groups – Instance-level firewall
NACL – Subnet-level firewall

4.Identity & Security
AWS IAM – User and role management
Policies – Define permissions
